# CausalIQ Repo Template User Guide

The causaliq-repo-template provides a starting point for new CausalIQ packages.
It provides:

- Standardised repo structure
- Skeleton command line interface (CLI)
- Python 3.9, 3.10, 3.11 and 3.12 virtual environments
- Pytest unit, functional and integration test framework with coverage reporting
- Continuous Integration test framework which is run locally and on Githib.
- Standardised documentation framework

## [Creating a new CausalIQ repo](template.md)

Full instructions on using this template to create a new repo for a new CausalIQ package are given [here](template.md)
